# Clase 1 – Fundamentos de TypeScript para Angular

## Objetivo
Comprender qué es TypeScript y por qué Angular lo utiliza.

## ¿Qué es TypeScript?
TypeScript es un superset de JavaScript que agrega tipado estático y herramientas para proyectos grandes.

## Tipos básicos
```ts
let nombre: string = "Angela";
let edad: number = 25;
let activo: boolean = true;
let skills: string[] = ["HTML", "CSS", "JS"];
```

## Interfaces
```ts
interface Usuario {
  nombre: string;
  email: string;
  activo: boolean;
}
```
