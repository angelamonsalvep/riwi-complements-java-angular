# Clase 3 – Componentes, Data Binding y Directivas

## Objetivo
Conectar lógica con la vista usando Angular.

## Interpolación
```html
<h1>{{ titulo }}</h1>
```

## Directivas
```html
<p *ngIf="activo">Activo</p>
<li *ngFor="let item of lista">{{ item }}</li>
```

Crea un componente Perfil con nombre y habilidades.
